Astara

Data & Computer Science Analyst Intern • Summer 2024

Visit Astara →

Overview

Astara is a global mobility provider focused on optimizing vehicle fleets and providing sustainable transportation solutions. As a Data & Computer Science Analyst Intern, I developed automation tools and data analysis solutions that significantly improved operational efficiency.

Key Accomplishments

  • Developed Python data analysis tools and automation solutions for fleet optimization
  • Created web scraping scripts to monitor over 1,000 competitors' prices for real-time market analysis
  • Automated the company's reporting system, reducing weekly report preparation time by over 90%
  • Built tools that continue to provide competitive insights and operational efficiency improvements

Technical Skills & Tools

Throughout this internship, I developed expertise in data engineering and automation:

  • Python programming for data analysis and automation
  • Web scraping technologies for competitive intelligence
  • Data pipeline development and ETL processes
  • Report automation and data visualization
  • API integration and data processing workflows

Impact & Learnings

This internship taught me the value of automation in business operations. By reducing manual reporting time by 90%, I freed up the team to focus on strategic analysis rather than data collection. The web scraping tools I developed provided real-time competitive intelligence that informed pricing decisions across the fleet.

I learned that effective engineering solutions must balance technical sophistication with practical usability. The tools I built needed to be maintainable by non-technical team members, which taught me the importance of clear documentation and user-friendly interfaces.
