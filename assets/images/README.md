# Image Assets

## Folder Structure

- **`hero/`** - Hero section images (your photo, background images)
- **`projects/`** - Project thumbnails and detail images
- **`experience/`** - Experience/work thumbnails and detail images

## Image Guidelines

### Recommended Sizes

- **Hero image**: 800x1000px (portrait) or 1200x800px (landscape)
- **Project/Experience cards**: 700x500px (landscape recommended)
- **Detail page images**: 1200x800px or larger

### File Formats

- Use `.jpg` for photos
- Use `.png` for graphics with transparency
- Use `.webp` for best performance (optional)

### Optimization

- Compress images before uploading
- Keep file sizes under 500KB for thumbnails
- Use descriptive filenames (e.g., `baja-sae-chassis.jpg`)

## Updating Images in HTML

Replace the `.image-placeholder` divs with:

```html
<img src="assets/images/projects/your-image.jpg" alt="Descriptive alt text">
```
